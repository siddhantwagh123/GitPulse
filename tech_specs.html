1. Technologies Used & Database Architecture

Frontend Stack

  • React & Vite: For blazing-fast dev reload and lightweight compiled output chunks.
  • Tailwind CSS: Full utility-first grid styling supporting class-based Light/Dark transitions.
  • Recharts: Handles SVG charts (Monthly aggregates, Weekly coding pattern bars, Top languages donut).
  • Lucide Icons: Provides lightweight SVG glyph keys.

Backend & Database

  • Node.js & Express: High-performance event loop gateway acting as the profile query middleware.
  • Dynamic API Data Layer: Powered by GitHub GraphQL & REST API integrations, fetching user details on the fly.
  • Data Caching: Local memory caches prevent rate limit triggers during active queries.
  • Local Storage: Persists user UI configurations (such as theme preferences).

2. UI/UX Engineering & Animations

The interface is designed with a premium, glassmorphic layout prioritizing readability and responsive feedback. Key UI mechanics include:

Interactive Animations

  • Spotlight Mesh: Compositor-accelerated radial background grid tracking mouse pointers.
  • Donut Hover Spotlight: Dynamic cells dim down to 40% opacity except the hovered language slice.
  • Count-Up Loaders: Dynamic 1-second percentage count-up timers driven by requestAnimationFrame.

Print Engine Optimization

  • Inset Box Shadow Colors: Calendar cells utilize inset shadow borders, forcing background colors to print even if browser print graphics are turned off.
  • Spacing Tables: Repeats HTML thead rows to act as margin headers on multi-page PDF splits.

When a user queries a handle, GitPulse intercepts the search to run an optimized GraphQL script fetching public timeline logs. The JSON data is processed client-side: summing languages mathematically to equal precisely 100.0%, organizing contributions into weekday/weekend buckets, calculating streaks, and matching logs to award gamified visual achievements (such as Weekend Warrior or Consistent Coder).
